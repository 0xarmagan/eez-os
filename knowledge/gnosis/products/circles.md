## Elevator pitch

An audacious experiment in peer-to-peer community currency. Users create and exchange their personal currency within a resilient, decentralised web of trust.

## Short description

A crazy experiment in money, built on trust and made to move. Anyone can create a currency via Circles and get value flowing within their community. The trust graph and inbuilt currency tooling support whatever else builders can imagine.

## Key differentiators

- Personal currency creation, peer-to-peer
- Trust graph as the backbone of transferability
- Hourly issuance per person
- v2 deployed on Gnosis Chain
- Native integration with Gnosis App
