# EEZ Quick Facts

The essentials about the Ethereum Economic Zone in 2 minutes.

---

## What is EEZ?

**The Ethereum Economic Zone** is a framework that solves Ethereum's fragmentation problem by enabling synchronous composability across layer-2 rollups.

---

## The Problem (In 1 sentence)

Every L2 is an isolated island. EEZ makes them part of one economic zone.

---

## The Solution (In 1 sentence)

Real-time zero-knowledge proofs enable atomic cross-chain transactions on Ethereum settlement.

---

## Key Facts

| Fact | Detail |

---

## The Core Vision

**"One Ethereum"**

Restore Ethereum as a coherent economic zone where:
- L2s compose together atomically
- Liquidity consolidates (not fragments)
- Smart contracts work across chains as if on one layer

---

## How It Changes User Experience

### Before EEZ
```
Aave (Ethereum) → Bridge → Uniswap (Arbitrum)
Multiple transactions | Slippage | Risk
```

### With EEZ
```
Aave (Ethereum) + Uniswap (Arbitrum) = Single atomic transaction
No bridge | No slippage | Ethereum settlement guarantees
```

## The Tradeoff

**Benefit:** Synchronous composability, atomic transactions, unified liquidity

**Tradeoff:** L2s must re-organize when Ethereum re-orgs (rare, manageable)

---

## Why It Matters

- **For Users:** Seamless cross-L2 interactions, better UX
- **For Developers:** Write once, compose everywhere
- **For Ethereum:** Regains gravity as coherent settlement layer
- **For L2s:** Access unified liquidity, stay specialized
