# Ethereum Economic Zone (EEZ) — Visual Identity System

## 1. Brand Positioning

**What it is:** A new canonical L2 framework for Ethereum — synchronously composable chains with shared liquidity and security. Born from Gnosis R&D with Jordi Baylina.

**Name:** Ethereum Economic Zone (EEZ)
**Pronunciation:** "ease" or "easy"
**Derivative terms:**
- EEZ rollups / EEZ-y rollups — chains built on the framework
- EEZ-y building — the developer experience
- EEZ-y liquidity — shared, not fragmented
- EEZ-y composability — synchronous, not bridged

**Relationship to Gnosis:** EEZ is infrastructure that serves all of Ethereum. Gnosis is a founding contributor, not the sole owner. The identity should feel like it belongs to Ethereum's commons — credibly neutral, technically serious, but with warmth and approachability baked into the name.

**Brand personality:** Confident but inviting. Where Gnosis is "playful subversive," EEZ is "calm enabler." It doesn't need to be loud — the name does the heavy lifting. The vibe is: *the hard part is already done, just build.*

## 2. Logo System

### Primary Wordmark

```
┌─────────────────────────────────────┐
│                                     │
│   E  E  Z                           │
│   ─────                             │
│   ETHEREUM ECONOMIC ZONE            │
│                                     │
└─────────────────────────────────────┘
```

- **EEZ** set in a custom geometric sans-serif — clean, wide-tracked, confident
- The two E's share a visual connection (mirrored, interlocking, or linked) to represent composability and synchronous connection between chains
- The Z has a subtle forward motion / flow to it — hinting at the "zone" as an active, expansive space
- **"ETHEREUM ECONOMIC ZONE"** sits below in a lighter weight as the descriptor

### Linked-E Motif

The signature visual device: **two interlocking or mirrored E's** that form a continuous shape.

```
  ┌───        ───┐
  ├───   ◄►   ───┤
  └───        ───┘
```

This represents:
- **Composability** — two chains that connect seamlessly
- **Shared liquidity** — value flowing between domains
- **Synchronous execution** — mirrored, simultaneous, atomic

The linked-E mark works as:
- A standalone icon / favicon
- A repeating pattern element
- An animated transition (the two E's sliding together)

### Sub-brand Lockup for Rollups

Individual EEZ rollups can adopt the system:


## 3. Color System

### Design Rationale

EEZ needs to feel like **Ethereum infrastructure** — not a Gnosis sub-brand. The palette anchors on a neutral blue that:
- Bridges Gnosis purple and Ethereum blue-grey
- Reads as technically credible and institutionally neutral
- Feels calm and expansive (like a "zone" you enter)

### Primary Colors

| Role | Name | Hex | Usage |
|------|------|-----|-------|
| **Primary** | Zone Blue | `#3B5FD9` | Logo, CTAs, primary UI elements |
| **Primary Dark** | Deep Zone | `#1A2B6B` | Backgrounds, dark mode, logotype on dark |
| **Primary Light** | Open Blue | `#7B9AEF` | Secondary UI, hover states, illustrations |

### Neutral System

| Role | Name | Hex | Usage |
|------|------|-----|-------|
| **Dark** | Midnight | `#0D1117` | Dark backgrounds, text on light |
| **Mid-dark** | Slate | `#21262D` | Cards on dark, secondary surfaces |
| **Mid** | Steel | `#8B949E` | Secondary text, borders |
| **Light** | Mist | `#E6EDF3` | Borders on light, subtle backgrounds |
| **White** | Clear | `#F8FAFC` | Primary light background |

### Accent Colors (sparingly)

| Role | Name | Hex | Usage |
|------|------|-----|-------|
| **Success / Flow** | Liquidity Green | `#2EA043` | Connected states, liquidity indicators |
| **Gnosis nod** | Soft Purple | `#8B7EC8` | Gnosis-adjacent contexts, "powered by" |
| **Warm accent** | Signal Amber | `#D4A72C` | Warnings, highlights, attention |

### Gradient

The signature gradient moves from **Deep Zone → Zone Blue → Open Blue**, left to right or radial — representing expansion from core Ethereum outward into the EEZ.

```
Deep Zone (#1A2B6B) ──────► Zone Blue (#3B5FD9) ──────► Open Blue (#7B9AEF)
```

For special moments (launch, hero sections), a subtle purple-to-blue gradient nods to the Gnosis lineage:

```
Soft Purple (#8B7EC8) ──────► Zone Blue (#3B5FD9)
```

## 4. Typography

### Display / Headlines

**Inter** (or **Space Grotesk**) — Bold / Black weight
- Geometric, modern, widely available
- Reads as infrastructure / developer tooling
- Wide tracking for "EEZ" lockup
- Open-source, web-native

### Body / Documentation

**Inter** — Regular / Medium weight
- Clean, highly legible at all sizes
- Excellent for technical documentation
- Already standard in Ethereum ecosystem tooling

### Code / Technical

**JetBrains Mono** or **Fira Code**
- For code snippets, technical specs, CLI output
- Ligatures optional

### Why not Gnosis fonts?

Boldonse (Gnosis display font) is too expressive and brand-specific for EEZ. EEZ should feel like shared infrastructure — closer to how Ethereum Foundation uses neutral, functional typography. Inter is the "Swiss Army knife" of the Ethereum ecosystem.

## 5. Iconography & Visual Language

### Zone Metaphor

The "zone" concept drives the visual system:

- **Concentric rings / ripples** — representing the expanding economic zone radiating from Ethereum
- **Connected nodes** — chains linked in synchronous composability
- **Flow lines** — liquidity moving freely between domains
- **The linked-E motif** — appearing in diagrams, backgrounds, patterns

### Technical Diagrams Style

```
┌─────────────┐     ┌─────────────┐
│  Ethereum   │◄───►│   EEZ-y     │
│   L1        │ ══  │   Rollup    │
└─────────────┘     └──────┬──────┘
                           │
                    ┌──────┴──────┐
                    │   EEZ-y     │
                    │   Rollup    │
                    └─────────────┘

═══  Synchronous composability
───  Shared liquidity
```

- Clean, minimal line diagrams
- Zone Blue for primary elements, Steel for secondary
- Liquidity Green for active connections / flow
- Animated versions show liquidity flowing between connected zones

### Pattern Library

- **Dot grid** — subtle background pattern suggesting coordinate space / territory
- **Concentric rings** — hero element, representing the zone expanding
- **Linked-E tessellation** — repeating pattern for backgrounds, merch, event materials

---

## 6. Motion & Animation Principles

- **Ease (literally)** — all animations use ease-in-out curves. Nothing is jarring. Everything flows.
- **Connection moments** — when showing composability, animate two elements sliding smoothly together
- **Ripple effect** — interactions create gentle ripples outward (the zone expanding)
- **Liquidity flow** — subtle animated gradients or particle flows showing value moving between chains

| **Technical** | Synchronous. Composable. Canonical. |
| **Aspirational** | The Ethereum Economic Zone is open. |
| **Playful** | Take it EEZ-y. |

### Headline Patterns

- "EEZ-y [noun]" — EEZ-y liquidity, EEZ-y composability, EEZ-y building
- "[Verb] with EEZ" — Build with EEZ, Scale with EEZ, Launch with EEZ
- "One Ethereum. [benefit]." — One Ethereum. Shared liquidity. / One Ethereum. No bridges.

### Tone

- **Calm confidence** — not hype, not aggressive
- **Technical credibility** — precise language, no hand-waving
- **Inclusive invitation** — "the zone is open" energy
- **Subtle wordplay** — the name inherently carries the pun; don't overdo it

## 9. Co-branding with Gnosis

### When Gnosis appears alongside EEZ:

- Gnosis uses its own brand system (purple, Boldonse, owl mark)
- EEZ uses its own brand system (blue, Inter, linked-E)
- They sit side by side with clear separation
- Connected by a "contributed by" or "founding contributor" label

```
┌──────────────────────────────────────────────┐
│                                              │
│  [EEZ mark]          [Gnosis mark]           │
│  EEZ                 GN🦉SIS                 │
│                                              │
│  "Gnosis is a founding contributor to EEZ"   │
│                                              │
└──────────────────────────────────────────────┘
```

### The Soft Purple accent (`#8B7EC8`) serves as a bridge:
- Used in EEZ materials when referencing the Gnosis relationship
- Creates visual continuity without making EEZ feel like a Gnosis sub-brand

## 10. What EEZ is NOT

- **Not a Gnosis sub-brand** — it's Ethereum infrastructure with Gnosis as a founding contributor
- **Not playfully subversive** — that's Gnosis's energy. EEZ is calm, credible, inviting
- **Not maximalist** — it serves all Ethereum builders, not a tribe
- **Not complex** — the whole point is that it makes things EEZ-y. The brand should feel effortless
